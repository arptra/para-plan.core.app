# Bloat при выключенном autovacuum

**Что не так:** Обновления/удаления накапливают мёртвые строки, таблица раздувается.

**Как проявляется:** Много чтения страниц, даже при индексах — медленно.

**Что обнаружит анализатор:** Советы: VACUUM/ANALYZE; включить autovacuum.

**Исправление:** VACUUM (FULL, ANALYZE) и вернуть autovacuum.

## Как запустить
```bash
docker compose up -d
./demo.sh
```
