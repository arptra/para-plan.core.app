# DISTINCT ON + сортировка без индекса

**Что не так:** Сортировка по (k, ts desc) на большой выборке.

**Как проявляется:** Долгая сортировка/временные файлы.

**Что обнаружит анализатор:** Совет: индекс (k, ts DESC).

**Исправление:** Создать покрывающий индекс.

## Как запустить
```bash
docker compose up -d
./demo.sh
```
