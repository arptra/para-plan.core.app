# Пролив сортировки во временные файлы

**Что не так:** ORDER BY по большой таблице, маленький work_mem → сортировка пишет во временные файлы.

**Как проявляется:** EXPLAIN показывает Sort с Memory/Disk, долгие запросы.

**Что обнаружит анализатор:** predicted.tempSpillRisk=2, рекомендации: индекс по ORDER BY или поднять work_mem.

**Исправление:** Добавить индекс по ts DESC, при необходимости увеличить work_mem.

## Как запустить
```bash
docker-compose up -d
./demo.sh
```
