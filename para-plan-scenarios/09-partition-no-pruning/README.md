# Нет отсечения партиций из-за функции на ключе

**Что не так:** Функция над партиционной колонкой мешает pruning.

**Как проявляется:** Сканируются многие партиции → медленно.

**Что обнаружит анализатор:** Совет REWRITE: убрать функцию с колонки.

**Исправление:** Использовать прямое сравнение ts с константой.

## Как запустить
```bash
docker compose up -d
./demo.sh
```
