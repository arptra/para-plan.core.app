# Исчерпание подключений (max_connections)

**Что не так:** Слишком много одновременных сессий.

**Как проявляется:** Новые соединения отвергаются.

**Что обнаружит анализатор:** Рекомендации по пулерам и лимитам.

**Исправление:** Внедрить пулер, ограничить pool size.

## Как запустить
```bash
docker compose up -d
./demo.sh
```
