# Неправильный порядок колонок в индексе

**Что не так:** Индекс (page_id, ts) не помогает данному запросу.

**Как проявляется:** SeqScan/Sort вместо Index Scan.

**Что обнаружит анализатор:** Совет: индекс (user_id, ts DESC).

**Исправление:** Создать подходящий составной индекс.

## Как запустить
```bash
docker compose up -d
./demo.sh
```
